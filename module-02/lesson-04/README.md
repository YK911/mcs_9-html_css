- Сімейства шрифту `serif` та `sans-serif`
- Підключення зовнішніх шрифтів через `GoogleFonts`
- Властивості оформлення шрифту:
  - `font-size`
  - `font-weight`
  - `font-style`
  - `font-family`
- Властивості оформелння тексту:
  - `text-align`
  - `text-decoration`
  - `text-transform`
  - `text-indent`
  - `line-height`
  - `letter-spacing`
  - `word-spacing`
  - `text-shadow`
- CSS змінні
